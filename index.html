Public Access Registration Database external API

Welcome to the Public Access Registration Database external facing API. Currently, this system it is still undergoing development.

Currently available endpoints

Below is a list of available endpoints and their uses:

Endpoint Description Example
/getAllManufacturers This endpoint will return an array containing JSONs of all of the registered manufacturers within PARD /getAllManufacturers
/searchManufacturersByName/{searchTerm} This endpoint takes a search term parameter, and returns an array containing JSONs of all of the registered manufacturers within PARD that match the search term /searchManufacturersByName/NIBSC
/searchManufacturersById/{manufacturer ID} This endpoint takes a manufacturer ID as a parameter, and returns a manufacturer that matches the provided ID. /searchManufacturersById/2
/getAllDevices This endpoint will return an array containing JSONs of all of the registered devices within PARD /getAllDevices
/searchDevicesByName/{searchTerm} This endpoint takes a search term parameter, and returns an array containing JSONs of all of the registered devices within PARD that match the search term /searchDevicesByName/medical device
/searchDevicesByManufacturer/{manufacturer ID} This endpoint takes the ID of a manufacturer and returns an array containing JSONs of all devices being produced by the manufacturer. /searchDevicesByManufacturer/2
/searchManufacturersByDevice/{searchTerm} This endpoint takes the GMDN_TERM_NAME of a device, and returns an array of JSONs containing all associated manufacturers of the device. /searchManufacturersByDevice/Rubella Virus IgM
